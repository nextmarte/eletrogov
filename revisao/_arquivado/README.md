# Arquivos arquivados

Estes rascunhos foram produzidos numa fase em que considerávamos extrair um artigo de revisão sistemática separado do artigo principal de extensão empírica do Vargas et al. (2021).

A decisão (2026-04-29) foi consolidar tudo no artigo principal, com a revisão entrando como **referencial teórico** dentro de `analise_egov.qmd`. Estes arquivos ficam aqui caso a ideia volte no futuro.

- `artigo_revisao_pt_v0.qmd` — rascunho em português, ~7 seções, ~55 papers
- `artigo_revisao_en_v0.qmd` — versão paralela em inglês

A infraestrutura ativa (`refs.bib`, `matriz_evidencia.csv`, `excerpts/`, `metodologia.qmd`) permanece em `revisao/` e alimenta o artigo principal.
